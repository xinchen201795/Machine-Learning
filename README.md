# 监督性学习和非监督性学习
有label的是监督性学习，反之为非监督性学习
## 分类，有features和label，学习的本质是找到features和label之间的关系，即mapping。
## 回归，跟分类一起构成监督性学习的两大板块。会给定函数及一些坐标点，与分类的差异在于最终的结果是连续的还是离散的。
## Clustering聚类分析，没有lable，只是简单看看新的点跟哪些类的距离比较近（先聚起来，根据聚类结果看属于什么）。
